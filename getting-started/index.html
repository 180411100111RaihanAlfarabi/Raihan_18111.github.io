    Chapter 1 : Errors in Numerical Computations¶

    ​ Ketika kita melakukan analisis numerik, biasanya ada dua sumber kesalahan, kesalahan pembulatan dan pemotongan. Kesalahan pembulatan disebabkan oleh ketidaktepatan dalam representasi bilangan real di komputer dan operasi aritmatika yang dilakukan dengannya. Sementara kesalahan pemotongan disebabkan oleh sifat perkiraan dari metode yang digunakan, biasanya dari penggunaan pendekatan di tempat prosedur matematika yang tepat, seperti yang kami gunakan deret Taylor untuk pendekatan fungsi. Sebagai contoh, kita dapat menggunakan deret Taylor untuk mendekati fungsi ex. $$ ex=1+x+x2/2!+x3/3!+x4/4!+ $$ Dengan lebih banyak istilah yang kita gunakan, perkiraan akan lebih dekat dengan nilai yang tepat. Mari kita gunakan Python untuk melihat contoh di atas.

    import numpy as np

exp = 0
x = 2
for i in range(10):
    exp = exp + \
       ((x**i)/np.math.factorial(i))
    print(f'Using {i}-term, {exp}')

print(f'The true e^2 is: \n{np.exp(2)}')

    Perkirakan kesalahan pemotongan

    ​ Kita dapat melihat urutan yang lebih tinggi yang kita gunakan untuk mendekati fungsi pada nilai, semakin dekat kita dengan nilai sebenarnya. Untuk setiap pesanan yang kami pilih, ada kesalahan yang terkait dengannya, dan perkiraan hanya berguna jika kami memiliki gagasan tentang seberapa akurat perkiraan tersebut. Inilah motivasi yang perlu kita pahami lebih dalam tentang kesalahan.

    Dari deret Taylor, jika kita hanya menggunakan n suku pertama, kita dapat melihat: $$ f(x)=fn(x)+En(x)=∑k=0nf(k)(a)(x−a)kk!+En(x) $$

    Chapter 2 : Numerical Solutions of Algebraic and Transcendental Equations¶

    ​ Diberikan fungsi f(x) pada bilangan mengambang x dan dua bilangan 'a' dan 'b' sedemikian rupa sehingga f(a)*f(b) < 0 dan f(x) kontinu di [a, b]. Di sini f(x) mewakili persamaan aljabar atau transendental. Temukan akar fungsi dalam interval [a, b] (Atau temukan nilai x sedemikian rupa sehingga f(x) adalah 0).

    Contoh :

    Input: Fungsi dari x, misalnya x3 - x2 + 2.
       Dan dua nilai: a = -200 dan b = 300 sedemikian rupa sehingga
       f(a)*f(b) < 0, yaitu, f(a) dan f(b) memiliki
       tanda-tanda yang berlawanan.
Output: Nilai root adalah : -1.0025
        ATAU nilai lain apa pun dengan diizinkan
        penyimpangan dari akar.

    Apa fungsi Aljabar dan Transendental? Fungsi aljabar adalah fungsi yang dapat direpresentasikan dalam bentuk polinomial seperti f(x) = a1x3 + a2x2 + ….. + e dimana aa1, a2, … adalah konstanta dan x adalah variabel. Fungsi transendental adalah fungsi non aljabar, misalnya f(x) = sin(x)*x – 3 atau f(x) = ex + x2 atau f(x) = ln(x) + x ….

    Apa itu Metode Bagi Dua? Metode ini disebut juga dengan metode interval halving, metode pencarian biner atau metode dikotomi. Metode ini digunakan untuk mencari akar persamaan dalam interval tertentu yang bernilai 'x' dimana f(x) = 0 . Metode ini didasarkan pada Teorema Nilai Antara yang menyatakan bahwa jika f(x) adalah fungsi kontinu dan ada dua bilangan real a dan b sedemikian rupa sehingga f(a)*f(b) 0 dan f(b) < 0) , maka dijamin memiliki setidaknya satu akar di antara mereka.

    Implementasi

    #The function is x^3 - x^2 + 2
def func(x):
    return x*x*x - x*x + 2

def bisection(a,b):

    if (func(a) * func(b) >= 0):
        print("You have not assumed right a and b\n")
        return

    c = a
    while ((b-a) >= 0.01):

        # Find middle point
        c = (a+b)/2

        # Check if middle point is root
        if (func(c) == 0.0):
            break

        # Decide the side to repeat the steps
        if (func(c)*func(a) < 0):
            b = c
        else:
            a = c

    print("The value of root is : ","%.4f"%c)

a =-200
b = 300
bisection(a, b)

    Output

    The value of root is : -1.0025

    Chapter 4 : Numerical Integration¶

    ​ Scipy menggunakan tiga metode untuk mengintegrasikan fungsi satu dimensi: trapesium (integrate.trapz), Simpson (integrate.simps) dan Romberg (integrate.romb). Metode trapesium (trapesium) adalah yang paling mudah dari ketiganya. Rumus trapesium sederhana menghitung integral fungsi f(x) sebagai luas di bawah kurva yang mewakili f(x) dengan mengaproksimasikannya dengan jumlah trapesium.

    Luas setiap trapesium dihitung sebagai lebar kali tinggi rata-rata. Contoh: Evaluasi integral:

    menggunakan aturan dasar trapesium. Kami akan menulis program kecil untuk mengevaluasi integral. Tentu saja kita harus memperkirakan jumlah trapesium yang akan digunakan; keakuratan metode kami tergantung pada angka ini.

    import math
#the function to be integrated:
def f(x):
    return x ** 4 * (1 - x) ** 4 / (1 + x ** 2)

#define a function to do integration of f(x) btw. 0 and 1:
def trap(f, n):
    h = 1 / float(n)
    intgr = 0.5 * h * (f(0) + f(1))
    for i in range(1, int(n)):
        intgr = intgr + h * f(i * h)
    return intgr

print(trap(f, 100))

    ​ Program integrasi dasar kami memiliki ketidaknyamanan tergantung pada jumlah trapesium yang harus kami ubah secara manual. Sebagai aturan dasar, jika kita menggandakan jumlah trapesium dan mendapatkan jawaban yang sama dalam 1 banding 1000000, jawabannya mungkin benar.

    Kegiatan 1:

    Jalankan programnya. Berapa banyak trapesium yang diperlukan untuk mendapatkan kesalahan relatif kurang dari 1 bagian dalam 1.000.000? Kita dapat melakukan ini sebagai berikut:

    from math import *
n = 1
while abs(trap(f, n) - trap(f, n * 2)) > 1e-6:
    n += 1
print(n)

    kegiatan 2 : menghitung

    Anda harus memodifikasi program sebelumnya karena jangkauan integrasi yang tidak terbatas. Periksa integrand untuk melihat di mana ia menjadi diabaikan. Kode yang dimodifikasi akan terlihat seperti ini.

    import math
#the function to be integrated:
def f(x):
    return math.exp(-x**2)

#define a function to do integration of f(x) btw. a and b:
def trap(f, n, a, b):
    h = (b - a) / float(n)
    intgr = 0.5 * h * (f(a) + f(b))
    for i in range(1, int(n)):
        intgr = intgr + h * f(a + i * h)
    return intgr

a = -10
b = 10
n = 100

while(abs(trap(f, n, a, b) - trap(f, n * 4, a * 2, b * 2)) > 1e-6):
    n *= 4
    a *= 2
    b *= 2

    Chapter 5 : Numerical Solution of System of Linear Algebraic Equations¶

    ​ Metode Gauss-Seidel adalah metode iteratif spesifik, yang selalu menggunakan nilai taksiran terbaru untuk setiap elemen di x. Sebagai contoh, pertama-tama kita asumsikan nilai awal untuk x2,x3,⋯,xn (kecuali untuk x1), dan kemudian kita dapat menghitung x1. Menggunakan x1 terhitung dan sisa x (kecuali x2), kita dapat menghitung x2. Kita dapat melanjutkan dengan cara yang sama dan menghitung semua elemen di x. Ini akan mengakhiri iterasi pertama. Kita dapat melihat bagian unik dari metode Gauss-Seidel adalah bahwa kita selalu menggunakan nilai terbaru untuk menghitung nilai berikutnya dalam x. Kami kemudian dapat melanjutkan dengan iterasi sampai nilai konvergen. Mari kita gunakan metode ini untuk memecahkan masalah yang sama yang baru saja kita selesaikan di atas.

    CONTOH: Selesaikan sistem persamaan linear berikut menggunakan metode Gauss-Seidel, gunakan ambang batas yang telah ditentukan sebelumnya ϵ=0,01. Ingatlah untuk memeriksa apakah kondisi konvergen terpenuhi atau tidak. $$ \begin{eqnarray*} 8x_1 + 3x_2 - 3x_3 &=& 14 \ -2x_1 - 8x_2 + 5x_3 &=& 5 \ 3x_1 + 5x_2 + 10x_3 & =& -8 \ \end{eqnarray*} $$

    a = [[8, 3, -3], [-2, -8, 5], [3, 5, 10]]


diag = np.diag(np.abs(a)) 


off_diag = np.sum(np.abs(a), axis=1) - diag 

if np.all(diag > off_diag):
    print('matrix is diagonally dominant')
else:
    print('NOT diagonally dominant')

    x1 = 0
x2 = 0
x3 = 0
epsilon = 0.01
converged = False

x_old = np.array([x1, x2, x3])

print('Iteration results')

for k in range(1, 50):
    x1 = (14-3*x2+3*x3)/8

    if dx < epsilon:

    Chapter 6 : Numerical Solutions of Ordinary Differential Equations¶

    Dalam buku catatan ini kita akan menggunakan Python untuk menyelesaikan persamaan diferensial secara numerik.

    import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline

    Persamaan orde pertama

    Mari kita coba persamaan diferensial biasa orde pertama (ODE), katakan: $$ \quad \frac{dy}{dx} + y = x, \quad \quad y(0) = 1. $$ Ini memiliki solusi bentuk tertutup $$ \quad y = x - 1 + 2e^{-x} $$ Kita akan menyelesaikan ini secara numerik.

    Pertama, mari impor modul "scipy" dan lihat file bantuan untuk fungsi yang relevan, "integrate.odeint",

    from scipy.integrate import odeint

# Define a function which calculates the derivative
def dy_dx(y, x):
    return x - y

xs = np.linspace(0,5,100)
y0 = 1.0  # the initial condition
ys = odeint(dy_dx, y0, xs)
ys = np.array(ys).flatten()

    # Plot the numerical solution
plt.rcParams.update({'font.size': 14})  # increase the font size
plt.xlabel("x")
plt.ylabel("y")
plt.plot(xs, ys);

    Bandingkan solusi numerik dengan solusi analitik dengan menunjukkan keduanya pada plot yang sama

    y_exact = xs - 1 + 2*np.exp(-xs)
y_difference = ys - y_exact
plt.plot(xs, ys, xs, y_exact, "+");

    Sekarang lihat perbedaan antara kedua seri:

    y_diff = np.abs(y_exact - ys)
plt.semilogy(xs, y_diff)
plt.ylabel("Error")
plt.xlabel("x")
plt.title("Error in numerical integration");
